# 📦 StockFlow – Smart Inventory Management System

StockFlow is a **web-based inventory management application** designed to help organizations efficiently track products, monitor stock levels, and manage inventory operations.

The system provides a **dashboard with analytics, product management features, stock alerts, and authentication**, allowing businesses to maintain accurate inventory data.

## ✨ Features

### 🔐 Authentication
- User Signup
- User Login
- Secure password hashing using **bcrypt**
- Session-based authentication
- Protected routes
- Logout functionality

### 📊 Dashboard
- Total products overview
- Total inventory quantity
- Low stock product detection
- Inventory health visualization
- Interactive charts using **Chart.js**

### 📦 Product Management
Users can:
- Add new products
- Edit product details
- Delete products
- Track SKU numbers
- Monitor product quantities
- Configure low stock thresholds

### ⚙️ Settings
- Default low stock threshold configuration
- Organization-based inventory management
- Customizable inventory alerts

### 🔔 Smart Alerts
Integrated **SweetAlert notifications** for:
- Login errors
- Access restrictions
- Product creation
- Product updates
- Product deletion
- Logout confirmation
